CSS padding

padding

padding 속성은 요소의 네 방향 안쪽 여백 영역을 설정합니다.

문법

padding : length | % |

정의

  • padding-top 속성은 요소의 상단 안쪽 여백을 설정합니다.
  • padding-right 속성은 요소의 우측 안쪽 여백을 설정합니다.
  • padding-bottom 속성은 요소의 하단 안쪽 여백을 설정합니다.
  • padding-left 속성은 요소의 좌측 안쪽 여백을 설정합니다.
  • padding : 10px 11px 12px 13px : 상단 10px 우측 11px 하단 12px 좌측 13px
  • padding : 10px 11px 12px : 상단 10px 우측, 좌측 11px 하단 12px
  • padding : 10px 11px : 상단, 하단 10px 우측, 좌측 11px
  • padding : 10px : 상단, 우측, 하단, 좌측 10px
