border-left-color border-left-color 속성은 요소의 왼쪽 테두리의 색상을 지정합니다.
border-left-style border-left-style 속성은 요소의 왼쪽 테두리의 스타일을 지정합니다.
border-left-width border-left-width 속성은 요소의 왼쪽 테두리의 너비를 지정합니다

border-right border-right 속성은 요소의 오른쪽 테두리의 너비, 색상, 스타일을 지정합니다.
border-right-color border-right-color 속성은 요소의 오른쪽 테두리의 색상을 지정합니다.
border-right-style border-right-style 속성은 요소의 오른쪽 테두리의 스타일을 지정합니다.
border-right-width border-right-width 속성은 요소의 오른쪽 테두리의 너비를 지정합니다.
border-spacing border-spacing 속성은 테이블의 테두리 간격을 설정합니다.
border-style border-style 속성은 요소 테두리 네 면의 스타일을 지정합니다.
border-top border-top 속성은 요소의 상단 테두리의 너비, 색상, 스타일을 지정합니다.

border-top-left-radius border-top-left-radius 속성은 요소의 좌측 상단 모서리를 둥글게 설정합니다.
border-top-right-radius border-top-right-radius 속성은 요소의 우측 상단 모서리를 둥글게 설정합니다.
border-top-style border-top-style 속성은 요소의 상단 테두리의 스타일을 지정합니다.
border-top-width border-top-width 속성은 요소의 상단 테두리의 너비를 지정합니다.

caret-color caret-color 속성은 input의 커서 색을 정의합니다.
clear clear 속성은 float 속성을 통해 태그를 부유시킨 이후 문서의 흐름을 제거하기 위해 쓰입니다.
clip clip 속성은 보이는 영역을 설정합니다.

column-fill column-fill 속성은 열의 지정방법을 정의합니다.

counter-increment counter-increment 속성은 콘텐츠의 순서 상태를 정의합니다.

empty-cells empty-cells 속성은 테이블의 빈요소의 속성을 설정합니다.
filter filter 속성은 그래픽 효과를 설정합니다.
flex flex 속성은 하나의 플렉스 아이템이 자신의 컨테이너가 차지하는 공간에 맞추기 위해 크기를 키우거나 줄이는 방법을 설정하는 속성입니다.

font font 속성은 글꼴을 설정합니다.
font-family font-family 속성은 선택된 요소에 우선 순위가 지정된 font family 이름과 generic family 이름을 지정할 수 있게 해줍니다.
font-size font-size 속성은 폰트의 크기(대문자 "M"의 크기)를 지정합니다. 폰트 크기를 바꾸면 em 과 ex <length> 단위로 계산된 다른 항목들의 크기를 바꿉니다.
font-size-adjust font-size-adjust 속성은 소문자를 기준으로 폰트 크기를 설정합니다.
font-stretch font-stretch 속성은 폰트의 굵기를 정의합니다.
font-style font-style 속성은 글꼴이 해당 font-family의 일반, 기울임꼴 또는 비스듬한 면으로 스타일 지정되어야 하는지 여부를 설정합니다.
font-variant font-variant 속성은 글꼴의 변형을 정의합니다.
font-weight font-weight 속성은 폰트(font)의 가중치(weight)나 굵기(boldness)를 명시한다. 몇몇 폰트들은 normal 나 bold 일 때만 가능합니다.
