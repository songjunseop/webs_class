CSS margin

margin

margin 속성은 요소의 바깥쪽 여백을 설정합니다.

문법

margin : length | % | auto

정의

  • margin-top 속성은 요소의 상단 바깥쪽 여백을 설정합니다.
  • margin-right 속성은 요소의 우측 바깥쪽 여백을 설정합니다.
  • margin-bottom 속성은 요소의 하단 바깥쪽 여백을 설정합니다.
  • margin-left 속성은 요소의 좌측 바깥쪽 여백을 설정합니다.
  • margin : 10px 11px 12px 13px : 상단 10px 우측 11px 하단 12px 좌측 13px
  • margin : 10px 11px 12px : 상단 10px 우측, 좌측 11px 하단 12px
  • margin : 10px 11px : 상단, 하단 10px 우측, 좌측 11px
  • margin : 10px : 상단, 우측, 하단, 좌측 10px
  • margin : auto : 자동값, 기본값(padding에는 없음)
  • margin : 0 auto : 가운데 정렬(블록구조)
